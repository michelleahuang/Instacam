# Instacam

[Instacam](https://the-instacam.herokuapp.com/#/login) is a full stack web application clone of Instagram where users can publicly share photos that can be viewed, liked by, and commented on by other users.

## Technologies 
* Ruby on Rails
* React/Redux
* PostgreSQL
* JavaScript
* HTML5 / CSS
